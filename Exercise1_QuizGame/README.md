## QUIZ GAME

### Steps Build:
```bash
go build
./Exercise1_QuizGame
```

### Flags:
* ```-csv```\
    data type: _string_\
    A csv filename in form of string to get question answers\
    Format for CSV file: question,answer (default "../resources/problems.csv")
* ```-l```\
    data type: _integer_\
    Time limit to answer a question (default 300 seconds)
